# Installer ReadMD

ReadMD est une petite application **gratuite** pour lire des fichiers Markdown
(`.md`) avec un rendu propre. Voici comment l'installer en 2 minutes.

> ℹ️ La 1re ouverture affiche un avertissement de sécurité **normal** : l'app
> n'est pas « signée » (la signature est payante). L'app est sûre et open
> source. Cet avertissement n'apparaît **qu'une seule fois** par ordinateur.

---

## 🍎 Sur Mac

1. Télécharge le fichier **`ReadMD_…_aarch64.dmg`** depuis la page
   [Releases](https://github.com/Axclgrd/ReadMD/releases).
2. Double-clique le `.dmg`, puis **glisse l'icône ReadMD dans le dossier
   Applications**.
3. **Première ouverture** (pour passer l'avertissement « développeur non
   vérifié ») :
   - Va dans  **Réglages Système → Confidentialité et sécurité**.
   - Descends jusqu'au message qui parle de « ReadMD » et clique
     **« Ouvrir quand même »**.
   - *(Variante : clic droit sur ReadMD dans Applications → **Ouvrir** →
     **Ouvrir**.)*
4. Voilà ! Les fois suivantes, un double-clic normal suffit.

## 🪟 Sur Windows

1. Télécharge le fichier **`ReadMD_…_x64-setup.exe`** depuis la page
   [Releases](https://github.com/Axclgrd/ReadMD/releases).
2. Lance-le. Si Windows affiche **« Windows a protégé votre PC »** :
   - Clique **« Informations complémentaires »** puis **« Exécuter quand
     même »**.
3. Suis l'installeur. ReadMD apparaît ensuite dans le menu Démarrer.

---

## Lire un fichier `.md`

- **Double-clique** un fichier `.md`, ou
- clic droit → **Ouvrir avec → ReadMD**, ou
- **glisse-dépose** le fichier dans la fenêtre, ou
- utilise le bouton **« Ouvrir »** dans l'app.

Bonne lecture ! 📄
